# Dungeon Crawler

## Setup

1. `git clone https://github.com/vincyboy1/my-dungeon-crawler.git`
2. `cd my-dungeon-crawler`
3. `rojo serve`
4. In Roblox Studio, use the Rojo plugin to connect to `localhost:34865`
